June Bootcamp

Courses overview

This is an immersive and intense four week program targeted for beginners. We cover everything from introductory HTML & CSS, to WordPress, Python, Ruby, web design and more. On the first week we will start by answering questions like "What is programming?", "Which programming language(s) should I learn first?". The camp will culminate in one big final project.

Intro to web development

The technologies taught in this course are the backbone of all websites, and knowledge of them is a necessity for things like web development, creating marketing emails, or even blogging! Our HTML & CSS workshops are designed to be a hands-on experience. During the sessions, scholars will create beautiful websites from scratch all while learning.

Intro to programming (python)

Python is a great language for beginners. Its beautiful and clean-looking syntax means you'll spend less time being confused (it looks sort of like English!), and more time understanding and applying the fundamentals of programming. You'll also be introduced to integers, strings, lists, dictionaries, objects and types
